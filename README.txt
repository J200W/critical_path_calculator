# critical_path_calculator
A python project that allows you to calculate the critical path of a graph. 

tables have the following structure :

 -------------- ---------- --------------
| num_activity | duration | predecessors |
 -------------- ---------- --------------
|      1       |    12    |              |
 -------------- ---------- --------------
|      2       |     4    |       1      |
 -------------- ---------- --------------
|      3       |     0    |       2      |
 -------------- ---------- --------------
 
graph : 1 --(12)--> 2 --(4)--> 3

Have fun !
